Prototype of a Finance Tracker app, which will be a social media app in which 
1. A user can sign up and login/logout.

2. A user can search for stock from the stock symbols.

3. A user can search for friends and can make friendships with them.

4. A user can view the stocks of his friends.

5. A user can search for his friends from his first name, last name and also from his email
